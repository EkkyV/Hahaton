# Factory of Numbers

Минимальный MVP-проект браузерной 2D-игры на **Vite + TypeScript + Phaser**.

## Команды
- `npm run dev` — запуск dev-сервера.
- `npm run build` — строгая проверка типов и production-сборка.
- `npm run preview` — локальный предпросмотр production-сборки.

## Запуск
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите проект:
   ```bash
   npm run dev
   ```
